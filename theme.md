BRUTALISM (or Brutalist Design)
Key Characteristics:
🖤 Stark Black & White

Zero gradients, pure contrast
Aggressive monochromatic palette
📐 Raw Geometric Forms

Heavy 2px and 4px borders
Sharp right angles, no soft curves
Grid patterns and harsh layouts
🔤 Extreme Typography

ALL CAPS, uppercase-only text
Extra-bold sans-serif (Cabinet Grotesk ExtraBold)
Monospace code styling throughout
Tight letter-spacing (-0.02em)
⚙️ Industrial/System Aesthetic

Looks like OS interface or terminal UI
Command-line style interactions
Technical jargon ("PROTOCOL_04", "NEURAL_ENGINE", "SYSTEM_STATUS")
Offset box-shadows mimicking mechanical depth
🎯 Anti-Design Philosophy

Stripped of decoration
Function over aesthetics
Raw, unapologetic, uncompromising
"Built for performance" mentality
Similar Design Movements:
Swiss Style (grid-based, minimal)
Industrial Design (raw materials aesthetic)
Cyberpunk/Dystopian UI (system-like interfaces)
90s Minimalism (extreme reduction)
This is intentionally aggressive and provocative—it rejects warmth and friendliness in favor of raw power and clarity. Perfect for a technical/enterprise SaaS brand! 💪
